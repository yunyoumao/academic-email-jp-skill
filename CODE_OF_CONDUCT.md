# Code of Conduct

This project is intended to support practical academic communication.

Participants are expected to:

- be respectful and constructive
- avoid sharing private or identifying information about students, professors, collaborators, reviewers, or institutions
- use synthetic examples when discussing sensitive scenarios
- focus criticism on wording, behavior, documentation, and maintainability

Reports or concerns can be raised through GitHub issues. Maintainers may edit, hide, or remove content that exposes private information or creates an unsafe contribution environment.
