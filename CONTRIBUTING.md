# Contributing

Thanks for your interest in improving this skill.

Good contributions include:

- clearer `SKILL.md` instructions
- synthetic example inputs and outputs
- scenario-specific checks for academic emails
- fixes for unnatural Japanese expressions
- documentation improvements

Before submitting a change, please check that:

- the use case is academic, research, university, or lab communication
- every example is synthetic and does not identify real people
- no private email, student ID, unpublished manuscript detail, or internal lab information is included
- the output avoids assistant meta language such as "this is a natural email"
- the tone is polite but not exaggerated or template-like

For new scenarios, include a short input file and an output draft under `examples/<scenario-name>/`.
